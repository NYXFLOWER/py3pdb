# py3pdb

Py3PDB is an underdeveloping python3 package for interacting with the RCSB Protein Data Bank (PDB) programmably, based on [Wbe Services](https://www.rcsb.org/pages/webservices).

Repository structure

```
├───docs
├───examples
├───py3pdb
│   ├───data
│   ├───search
│   ├───download
│   └───utils
```

## installation

```
pip install py3pdb
```
